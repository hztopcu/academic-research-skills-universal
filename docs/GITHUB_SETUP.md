# GitHub Repository Setup

## Actions

The repository includes `.github/workflows/tests.yml`. To get a green check on GitHub:

1. Open `Actions`.
2. Enable workflows if GitHub asks for confirmation.
3. Open the `Tests` workflow.
4. Run the workflow from `main`, or push a new commit.
5. Wait for all Python matrix jobs to pass.

The README badge uses:

```md
[![Tests](https://github.com/hztopcu/academic-research-skills-universal/actions/workflows/tests.yml/badge.svg)](https://github.com/hztopcu/academic-research-skills-universal/actions/workflows/tests.yml)
```
