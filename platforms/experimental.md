# Experimental Platforms

These platforms have installer and verification coverage, but they are still under testing:

- Agent Skills
- Gemini CLI
- OpenCode
- CodeBuddy
- Kilo Code
- GitHub Copilot CLI
- OpenClaw
- Factory Droid
- Trae
- Trae CN
- Hermes
- Kimi Code
- Amp
- Kiro IDE/CLI
- Pi coding agent
- Devin CLI
- Google Antigravity

Most experimental platforms currently receive the portable bundle layout. Promotion to stable should require:

- A documented platform-specific usage path.
- Automated install and verification coverage.
- Manual end-to-end validation inside the actual platform.
- Notes about any missing orchestration features.
